About

DotA Pro was created as Tommy Blanchard's capstone project for The Data Incubator program. A logistic regression model was trained on the results of hundreds of thousands of ranked DotA 2 matches, and will calculate the probability of winning based on input heros and lanes.

The code for the modeling can be seen here, and the code for the app itself can be seen here.

500GBs of data was taken from Open Dota data dumps. This was parsed and processed using Python. The modeling was done using scikit-learn and visualizations created using bokeh. Flask was used as the framework.
